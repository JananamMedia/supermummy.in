# SuperMummy Website

**supermummy.in** — India's leading Hindi-language pregnancy health YouTube channel.

Built by [Jananam Media Pvt. Ltd.](https://jananam.co), Goa.

## How to Update

Open `data.js`, change the numbers, save, push to GitHub. Two minutes. Never touch the design.

## Files

| File | Purpose |
|------|---------|
| `index.html` | About page (homepage) |
| `work-with-us.html` | Partners / brands / media page |
| `mummies.html` | Hindi page for pregnant women |
| `data.js` | All updatable numbers — edit this monthly |
| `style.css` | Shared styles |
| `sitemap.xml` | For Google Search Console |
| `robots.txt` | Search engine directives |
| `CNAME` | GitHub Pages custom domain |
| `media-kit.pdf` | Downloadable one-page media kit |
| `logo.png` | SuperMummy logo (to be added) |

## Monthly Update Checklist

1. Open `data.js`
2. Update: `totalViews`, `monthlyViews`, `subscribers`, `totalVideos`, `lastUpdated`
3. Optionally update `growthData` array and `topCities`
4. Save and push
